# CRUD Project using Django

This project implements a CRUD (Create, Read, Update, Delete) functionality using Django, a high-level Python web framework. It allows users to perform basic database operations on a specified model.

## Table of Contents

- [Features](#features)
- [Requirements](#requirements)
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)

## Features

- **Create:** Add new entries to the database.
- **Read:** Retrieve and display existing entries from the database.
- **Update:** Modify existing entries in the database.
- **Delete:** Remove entries from the database.

## Requirements

- Python 3.x
- Django 3.x

## Usage

1. Start the Django development server:

    ```bash
    python manage.py runserver
    ```

2. Open your web browser and go to `http://127.0.0.1:8000/` to access the application.
